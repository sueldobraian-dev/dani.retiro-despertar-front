# Retiro Despertar - Frontend

Bienvenido al repositorio frontend del proyecto **Retiro Despertar**. Esta aplicación está diseñada para ofrecer una interfaz moderna, interactiva y premium para la gestión y presentación del retiro.

---

## 📂 Documentación del Proyecto

Encuentra a continuación las referencias a los documentos clave del proyecto ubicados en la carpeta `docs`:

*   📖 **[Instrucciones de Uso (INSTRUCCIONES.md)](docs/INSTRUCCIONES.md)**: Guía detallada para la configuración local del entorno de desarrollo, instalación de dependencias y ejecución de comandos principales.
*   📐 **[Arquitectura (ARCHITECTURE.md)](docs/ARCHITECTURE.md)**: Documentación técnica sobre la estructura del proyecto, flujo de datos y decisiones de diseño arquitectónico.

---

## 🛠️ Estructura del Repositorio

El proyecto está organizado de la siguiente manera:

*   **`docs/`**: Contiene la documentación técnica y de desarrollo del proyecto.
*   **`lib/`**: Directorio destinado a funciones de utilidad, helpers y lógica compartida.
*   **`public/`**: Recursos estáticos como imágenes y assets de la aplicación.
